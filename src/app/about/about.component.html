About/Help

Class: a class is a list of students. You can have as many classes as you want by creating them in the `Add Class` tab.

Students: to create a list in a selected class, simple add them on the `Add Student` tab. You can add them individually or as a list separated by a comma and then click Add Student button. ex: `Matt` or `Matt,Bobby,Kenny,Rick`

Edit: to edit the selected class's students, click the edit button (Top bar next to about/help). This will allow editing of that class's students.

Randomizing: to randomize a list of students. select the Randomize button (Top next to Edit). This will allow you to randomize either by single or in groups of 2, 3, or 4.
